# Changelog: CORE-002 - Schema Update: Advanced Dependencies

## Changes
- Updated `backend/app/schemas/task.py`:
    - Added `DependencyBase`, `DependencyCreate`, and `Dependency` Pydantic models.
    - Added `is_milestone` and `deadline_at` to `SubtaskBase`, `SubtaskShortCreate`, `TaskBase`.
    - Updated `TaskInDBBase` to include `blocked_by` and `blocking` as lists of `Dependency` objects.
- Fixed a database schema mismatch bug by creating and running `backend/mcp_sync_db.py` to add missing columns (`is_milestone`, `deadline_at`).

## Verification
- Executed `backend/tests/test_schemas_standalone.py` which confirmed Pydantic validation works for new fields and dependency objects.
- Manual execution of `mcp_sync_db.py` inside the container confirmed schema sync.
