# INT-03 · Magic Breakdown UI Button

**Timestamp**: 18:45:00 Local Time  
**Author**: Ralph  

## Why
Users often need help breaking down large, complex tasks into manageable subtasks. Integrating an AI decomposition feature directly into the task form provides a "magic" shortcut to better project structure.

## What Changed
- `backend/app/schemas/task.py`: 
    - Added `subtasks` field to `TaskUpdate` to support batch creation of children during task edits.
    - Added `description` field to `TaskShortCreate` for complete subtask metadata.
- `backend/app/crud/crud_task.py`: Updated `update` method to handle and create nested subtasks if provided in the update payload.
- `frontend/src/components/task-form.tsx`:
    - Added a "Breakdown" tab to the task form.
    - Implemented a "Magic Breakdown" button that calls the `/ai/decompose` API.
    - Integrated `useFieldArray` from `react-hook-form` to manage a list of suggested (and editable) subtasks.
    - Added UI to review, edit, and remove suggested subtasks before saving the main task.

## Notes
- The "Magic Breakdown" uses the current task's title and description as context for the AI.
- Subtasks added in the "Breakdown" tab are created as actual database records only when the "Save Task" button is clicked.
- Works for both task creation and task editing.
