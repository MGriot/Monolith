# Changelog: Backend Core Implementation

**Date:** 2026-01-27
**Task IDs:** BACK-002, BACK-003, BACK-004

## Changes
- **Database Models:** Implemented SQLAlchemy models for `User`, `Project`, `Task`, and `Subtask` with full metadata and relationships.
- **Authentication:** Established JWT-based auth system with `OAuth2PasswordBearer`, password hashing (bcrypt), and dependency-based current user retrieval.
- **Projects API:** Created full CRUD endpoints for Project management (Create, Read, Update, Delete) with owner-based filtering.
- **Base CRUD:** Implemented a generic `CRUDBase` class to streamline future API development.
- **Security:** Added security utilities for token generation and verification.
