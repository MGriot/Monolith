## Why
The user decided to pivot away from OpenAI/Anthropic-powered "Magic Breakdown" features in favor of deeper Model Context Protocol (MCP) integration. This task removes the redundant AI code to keep the system lean and focused on standard control.

- `backend/app/core/config.py`: Removed LLM-specific settings.
- `frontend/src/components/task-form.tsx`:

    - Removed `handleDecompose` and all AI-related logic.
