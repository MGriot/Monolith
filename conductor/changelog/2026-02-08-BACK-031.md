## Changes
- Modified `backend/app/api/deps.py`:
    - Defined `reusable_oauth2_optional` using `OAuth2PasswordBearer` with `auto_error=False`.
    - Updated `get_current_user_optional` to use this new optional scheme.
- This ensures that endpoints using `get_current_user_optional` (like user registration) do not automatically return a `401 Unauthorized` when the `Authorization` header is missing.
